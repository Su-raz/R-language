##9. Interpret the result of the correlation matrix plot obtained above with respect to the correlation coefficient to be used for each pair
#Ans:In the correlation matrix plot obtained above, the diagonal represents the correlation of each variable with itself, which is always 1. The upper triangle of the plot is a mirror image of the lower triangle, which contains the actual correlation coefficients. The color of each cell indicates the strength of the correlation, with darker colors indicating stronger correlations.
#In the plot, we can see that mpg and disp have a negative correlation, while mpg and cyl have a positive correlation. This suggests that cars with higher mpg tend to have lower displacement and fewer cylinders. On the other hand, cars with more cylinders tend to have higher displacement and lower mpg.
